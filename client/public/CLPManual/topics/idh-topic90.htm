Offer

Offer is some thing that the hotel or some other company will give for a customer who use their product or service to make them happy and make their customer more closer to the hotel beside the quality product.the offer might be in the form of the discount of price,free product or service or any other mean.In this system the offer is in the form of price discount of the service with selected product in specified date interval.

 

There are two user interface in the Offer

                    1.Offer List Page

                    2.Offer adding page

User interface will be like this...

 

Offer List

this is the list of the Offers registered by the branch admin of the system

 

Selection

This is a selection icon for selecting a lists whether single select of the offer or multi select by switching to the right for delete or update purpose if you choose single select the you can update or delete single one from the list

 

Add Offer

This button is used for entering to the offer adding page  by clicking

 

Search

This is the search text field for the offer that you added before.In here you can write any thing you want to filter either entering offer name or description or if anything that you write is included inside the list  then it will filter and display that to you.if you clear the text field the the table will be display all list

 

Update & Delete

The first icon used to update the offer added before like change the name of the offer title,Service,discount rate and the second icon use to delete the offer that you added
