Branch

One of the task of the super admin in the loyalty system is that registering the branch in detail, update registered branch,delete registered branch you can also search the specified branch finally  for that branch assign admin to administer under that branch.on the other hand we can delete and shift the branch admin

List of Branches

In here you will see the list of branches registered to loyalty system first list will be 10 but you can list 15 or 20 as you choice

 

Selection Choice

It is the selection of branches to delete or update we can select multi for all or some but if we choose single select then only one branch can be update or delete

 

Add Branch

By using this button  and filling important information  exactly with out making all fields empty than clicking save button to register the branch and finally you will see the registered branches in the list.

 

Search

Your can search for branches that you want from the registered,by entering the name or city,region,telephone

 

Update and Delete Branch

Selecting the branches that you want to update or delete then by Clicking the first icon you can update and also by clicking the second red icon to delete registered branch 

 

Shift Admin

First to Shift the assigned admin to selected branch you have to select the branch admin than click shift admin and also you can delete admins those are unassigned by clicking delete admin
