Adding level rule page will be like this:

Level Name

This is the name that the branch admin will set for the rule like gold ,silver...

 

Level Description

This is the description of the level rule that the branch admin will set

 

Minimum and maximum point

This range is that the admin will use for the set of the rule.if a customer fulfill the minimum point and maximum point or between this then that customer will be categorized under specified category.

 

Save Rule

After filling all required field the click the save level button to set the rule
