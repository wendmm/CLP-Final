CUSTOMER

 

This is the customer list page that the loyal customer will be displayed with detail things:

Customer List

This will display the list of customer registered as loyal either be refereed of by itself

 

Selection

This switch box will be used for multi select or single select of  the customer that we want to update or delete if we want to delete more than one customer then switch to the left and allow multi select

 

Search

This is the search text field for the customer those are members of this loyalty program.In here you can write any thing you want to filter either entering customer name or phone number or other  then it will filter and display that to you.if you clear the text field the the table will be display all customer list

 

Update and Delete

The first icon used to update the the customer status and the second icon use to delete the customer those are members in the loyalty program.
