LEVEL

 

In customer loyalty program this level module is included.In this level module Admin can set rule for customers to differentiate one from the other according to the transaction customers did In order to give like some rewards and also discounts maybe according to the level from point perspective  by setting minimum and maximum point rule.

 

There are two user interface in the level

                    1.Level List Page

                    2.Level Rule adding page

 

so the interface below will be the level list page

Selection

This is a selection icon for selecting a lists whether single select of the level or multi select by switching to the right for delete or update purpose if you choose single select the you can update or delete single one from the list

 

Add Level

This button is used for entering to the level rule adding page  by clicking

 

Search

This is the search text field for the rule that you gave before.In here you can write any thing you want to filter either entering level name or description or if anything that you write is included inside the list  then it will filter and display that to you.if you clear the text field the the table will be display all list

 

Actions

The first icon used to update the rule like change the name of the level ,description, minimum point, maximum point and the second icon use to delete the rule that you add

 

Level list

this is the list of the levels registered by the branch admin of the system
